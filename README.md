# Quart

Tonic is a Dart library that models components of music theory.


## Installing via Pub

Add this to your package's `pubspec.yaml` file:

    dependencies:
      tonic: any


## Example Usage

    import "package:tonic/tonic.dart";

    main() {
    }

More examples can be found in the tests in tonic_tests.dart.
